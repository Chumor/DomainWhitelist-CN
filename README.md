# DomainWhitelist-CN
常用CN网站白名单，纯列表，用于 NekoBox，控制绕过代理的网站。

## 版本说明
* DomainWhitelist-CN_
* Core只含有热门网站
* all含有白名单全部网站
* 无后缀为我的特供版


## 使用方法
* NekoBox-NekoBox-工具-备份-从文件中导入（该功能为覆盖导入请注意备份你的路由）

**其他方法自行摸索**
